# 🎯 Online Quiz Application

🔗 Live Demo: https://fastidious-malasada-a026e9.netlify.app/

A multi-step **Quiz Application** built using HTML, CSS, and JavaScript.  
The app includes a timer-based quiz system and visual result representation using Highcharts.

---

## 🚀 Features

✅ Multi-step form navigation  
✅ Personal details collection  
✅ Confirmation screen before quiz  
✅ Test selection system  
✅ Quiz instructions page  
✅ ⏱️ Countdown Timer  
✅ Dynamic score calculation  
✅ 📊 Score visualization using Highcharts Gauge Chart  
✅ Result breakdown with correct & selected answers  
✅ Fully deployed on Netlify  

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & Layout  
- **JavaScript (Vanilla JS)** – Logic & Functionality  
- **Highcharts.js** – Score Gauge Visualization  
- **Netlify** – Deployment  

---

## 📊 Highcharts Integration

The result page uses a **Highcharts Gauge Chart** to visually display:

- Total score
- Maximum points
- Performance range (Red → Yellow → Green)
- Real-time score update

Highcharts Official Website:  
https://www.highcharts.com/

---

## ⏱️ Timer Functionality

- Countdown timer starts when quiz begins
- Auto-submit on time completion
- Prevents further interaction after submission

---

## 🧠 How It Works

1. Enter personal details
2. Confirm information
3. Select test
4. Read instructions
5. Start quiz
6. Timer runs
7. Submit answers
8. View:
   - Score gauge
   - Correct vs Selected answers
   - Total points
